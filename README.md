# Godius Eternal War キャラクターシミュレーター

オンラインゲーム「Godius Eternal War」のキャラクター育成をシミュレーションするWebツールです。職業・レベル・パラメータ・スキル・習得魔法などを入力すると、必要な「力の玉」の量、到達可能レベル、装備性能などを計算して表示します。

## 主な機能

- **キャラ診断**：職業・副業・レベル・各パラメータ（STR/INT/DEX/AGR/VIT/MEN）・スキル・習得魔法から、力の玉の使用内訳と過不足を診断します。
- **必要玉数シミュレーター（リアルタイム）**：入力変更を監視し、必要玉数の内訳・合計、到達可能レベル、現在レベルとの過不足を即時に再計算して表示します。
- **スキル必要玉数の計算**：スキルアップテーブルに基づき、確率期待値方式・固定方式を併用（レベル17までは期待値、18以降は固定）して累計玉数を算出します。
- **装備シミュレーター**：装備品とパラメータから WC・HC・AC・DC・重さを計算します。
- **おまけ情報**：魔法詠唱時間や MP/SP 回復などの補助情報を表示します。
- **セーブ／ロード**：入力内容を localStorage に保存・復元します。
- **共有URL**：現在のキャラ状態をURLに埋め込んで共有できます。

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルドやサーバーは不要です（静的HTML/CSS/JavaScript）。

1. 職業・副業・レベルを選択します。
2. パラメータ・スキル・習得魔法を入力します。
3. 診断結果や必要玉数がリアルタイムに表示されます。
4. 必要に応じてセーブや共有URLの生成を行います。

## 技術構成

- HTML 4.01 / CSS / 素の JavaScript（フレームワーク非依存）
- 外部ライブラリなし
- データ永続化：ブラウザの localStorage
